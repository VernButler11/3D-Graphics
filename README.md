# 3D-Graphics
CS 330 Final Project – 3D Scene
Overview
This repository contains my final project for CS 330: Computational Graphics and Visualization. The project demonstrates my ability to design, model, and render a complete 3D scene using OpenGL. It includes custom‑built objects, hierarchical transformations, lighting, textures, and camera controls. The scene is based on a reference image and recreated through iterative design and development.
This repository also serves as part of my Computer Science program portfolio.

Included Artifacts
• 	3D Scene ZIP Folder – Complete project source code
• 	Design Decisions Document – Explanation of the design and development process
• 	README (this file) – Reflection and project summary

Project Features
• 	Custom 3D objects constructed from primitive shapes
• 	Hierarchical transformations for accurate object positioning
• 	Ambient, diffuse, and specular lighting
• 	Texture mapping and material adjustments
• 	Camera movement and scene navigation
• 	Modular, readable rendering structure

Design Approach
I approached the project by breaking the scene into smaller components and planning how each object and transformation would fit together. I used an iterative design process: starting with basic shapes, refining proportions, adding textures, and adjusting lighting until the scene aligned with the reference image. This structured approach helped maintain clarity and consistency throughout development.

Development Approach
My development process focused on modularity and incremental testing. I separated rendering logic, transformations, and object construction into dedicated functions. Iteration played a major role—each milestone allowed me to refine lighting, camera behavior, and object placement. Over time, my code became more organized, maintainable, and aligned with best practices.

Reflection
This project strengthened my understanding of computational graphics, spatial reasoning, and visual modeling. Working with transformations, lighting, and scene composition gave me practical experience that supports both my academic goals and future professional work. These skills are valuable in fields involving simulation, visualization, engineering, and interactive software.
